Attempt

A website that can display latest records, stats and live scores built using Node.js ,MongoDB and Bootstap.


Features

This website can show details about live, upcoming and recently concluded matches. A separate section for women's cricket has also been provided. User can also create an account, build a small profile and search for cricket records. Searched records can be found in favourites. Latest news and plethora of statistics can be found on the website. The website is mobile-responsive as well.
